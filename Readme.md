# Notes (console App)

## О приложении
Консольное приложение на Kotlin для создания «архивов» и заметок внутри них.
Позволяет создавать архивы, добавлять заметки, просматривать список архивов и содержимое заметок
через простое меню в терминале.

## Как запустить
Правой кнопкой по `Main` → **Run 'Main.kt'** (или Ctrl+Shift+F10).

## Краткое руководство по использованию
1. При запуске появляется главное меню:
    - 1 — Создать архив
    - 2 — Перейти к вашим архивам
    - 0 — Выход
2. Внутри архива можно добавить заметку или просмотреть список заметок.
3. Ввод проверяется на пустую строку.
